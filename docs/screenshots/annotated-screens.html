  8. 8 Par stepper — teal ∧∨ arrows; default Par 3 for all holes; adjustable per hole (typically 3–5 for disc golf)

2
3
4

  1. 1 Cancel — dismisses without persisting any changes to the course record
  2. 2 "Edit Course" nav title — distinguishes this modal from "New Course" to prevent user confusion
  3. 3 Save (active) — teal because the form already has a valid name; tap to persist changes to the local SwiftData store and sync to CloudKit
